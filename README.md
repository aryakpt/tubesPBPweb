# Aurora

Aurora merupakan  web apps blog sederhana. Pengembangan yang dilakukan masih menggunakan PHP native.

## Installation

1. Clone repository ini

```bash
git clone https://github.com/aryakpt/tubesPBPweb.git
cd tubesPBPweb
```
